MyFlix Documentation

The web application will provide users with access to information about different movies, release dates, rating, and run times. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

HTTP Request-Responce Cycle

Business Logic URL HTTP Method Request Body Data Format Response Body Data Format
Return a list of ALL movies to the user /movies GET None A JSON object holding data about all the movies
Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user /movies/:Title GET None A JSON object holding data about a single movie, containing description, genre, director, image URL, whether it’s featured or not. Example:
{
"Title": "The Shawshank Redemption",
"Description":"Over the course of ...",
"Release date": "1994",
"Rating": "R",
"Run time": "2h 22m",
"Genre":{
"Name":"Drama",
"Description":"Drama films ..."
},
"Director":{
"Name":"Frank Darabont",
"Bio":"Three-time Oscar nominee ...",
"Birth":"January 28, 1959",
"Death":"NA",
},
"ImageURL":"",
"Featured":false
},
Return data about a genre (description) by name/title /movies/Genre/:genreName GET None A JSON object holding data about a genre (description) by name/title. Example:
{
"Name":"Drama",
"Description":"Drama films ..."
},
Return data about a director (bio, birth year, death year) by name /movies/director/:directorName GET None A JSON object holding data about a director (bio, birth year, death year) by name. Example:
{
"Name":"Frank Darabont",
"Bio":"Three-time Oscar nominee ...",
"Birth":"January 28, 1959",
"Death":"NA",
},
Allow new users to register /users POST A JSON object holding data about the user to add, sturctured like:
{
name:"Emma",
favoriteMovies: [ ]
} 		A JSON object holding data about the user that was added, including an ID:
{
id:3,
name:"Emma",
favoriteMovies: [ ]
}
Allow users to update their user info (username) /users/:Username PUT None A text message indicating the user info that was updated
Allow users to add a movie to their list of favorites (showing only a text that a movie has been added) /users/:Username/movies/:MovieID POST None A text message indicating the user's list of favorite movies was updated
Allow users to remove a movie from their list of favorites (showing only a text that a movie has been removed) /users/:Username/movies/:MovieID DELETE None A text message indicating the user's list of favorite movies was updated
Allow existing users to deregister (showing only a text that a user email has been removed) /users/:Username DELETE None A text message indicating the user info that was successfully removed
